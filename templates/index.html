To use this app, you need to set up a Genius access token:

  1. Go to Genius API Clients and create an account if you don't have one
  2. Create a new API client
  3. Copy your Client Access Token
  4. Create a file named .env in the project root directory
  5. Add the following line to the file: GENIUS_ACCESS_TOKEN=your_token_here
  6. Restart the application
